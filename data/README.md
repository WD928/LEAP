# Hot-Start Additive Data

`hot_start_additives.csv` is the main table. It combines molecule identifiers, measured PCE values, relative PCE change, hard molecular descriptors, and soft mechanistic descriptor means/standard deviations.

Soft descriptors were aggregated from repeated model outputs with `N = 10` and `temperature = 0.7`. See `hot_start_data_dictionary.json` for column definitions.
